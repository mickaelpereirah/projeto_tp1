Detailed Description

Arquivo de inclusão central para todos os containers do sistema.

Este arquivo header agrupa todos os containers do sistema em um único ponto de inclusão, facilitando o uso dos containers em outros módulos. Basta incluir este arquivo para ter acesso a todos os containers disponíveis.

Containers incluídos:

  • ContainerReserva: Gerencia reservas
  • ContainerHospede: Gerencia hóspedes
  • ContainerHotel: Gerencia hotéis
  • ContainerGerente: Gerencia gerentes
  • ContainerQuarto: Gerencia quartos
