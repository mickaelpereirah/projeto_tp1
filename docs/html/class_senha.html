Detailed Description

Representa uma senha segura de exatamente 5 caracteres.

A classe Senha é responsável por armazenar e validar senhas seguras conforme regras rigorosas de segurança. A validação inclui:

  • Comprimento fixo de exatamente 5 caracteres
  • Caracteres permitidos: letras (a-z, A-Z), dígitos (0-9) e caracteres especiais (!"#$%&?) - Pelo menos uma letra minúscula (a-z) - Pelo menos uma letra maiúscula (A-Z) - Pelo menos um dígito (0-9) - Pelo menos um caractere especial (!"#$%&?)
  • Letra não pode ser seguida por letra
  • Dígito não pode ser seguido por dígito

Exemplos de senhas válidas:

  • A1b#2
  • p@Q3r
  • X9!y8
