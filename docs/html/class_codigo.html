Detailed Description

Representa um código identificador válido.

A classe Codigo é responsável por armazenar e validar códigos utilizados no sistema para identificação de entidades.

As regras de validação típicas podem incluir:

  • Formato específico (ex: AAA000)
  • Tamanho fixo ou variável
  • Caracteres permitidos (letras, números)

A implementação específica das regras de validação é definida no método validar() privado.
