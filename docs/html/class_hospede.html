Detailed Description

Classe que representa um hóspede no sistema de gestão de hotéis.

Esta classe herda de Pessoa e adiciona atributos específicos para hóspedes. Um hóspede pode fazer reservas em hotéis e possui endereço e cartão de pagamento.

Atributos herdados de Pessoa:

  • nome: Nome do hóspede
  • email: Email do hóspede

Atributos específicos:

  • endereco: Endereço do hóspede (tipo Endereco)
  • cartao: Cartão de pagamento do hóspede (tipo Cartao)

Relacionamentos:

  • Herda de Pessoa (1:1)
  • Faz Reserva (0:* para 1)
