Detailed Description

Container para armazenar e gerenciar hotéis do sistema.

Esta classe implementa um container para armazenamento em memória de objetos do tipo Hotel. Utiliza uma estrutura de unordered_map onde a chave é o email do gerente e o valor é um vector de hotéis, permitindo que cada gerente possua múltiplos hotéis (relacionamento 1:N).

Características principais:

  • Um gerente pode ter vários hotéis
  • Cada hotel possui código único no sistema
  • Busca por gerente retorna todos os seus hotéis
  • Busca por código retorna hotel específico
  • Permite remoção de hotel individual ou de todos os hotéis de um gerente

Estrutura de dados:

unordered_map<string, vector<Hotel*>>
Chave: email do gerente
Valor: vector com todos os hotéis deste gerente

Atualiza os dados de um hotel existente.

Parameters
hotelPonteiro para o hotel com dados atualizados
Returns
true se atualizado com sucesso, false se não encontrado

Busca o hotel pelo código e substitui o ponteiro no vector. O código do hotel NÃO pode ser alterado (é a chave primária).

IMPORTANTE: O ponteiro antigo não é liberado. Se necessário, faça delete antes de atualizar para evitar memory leak.

Exemplo de uso:

Hotel* h = container.pesquisarPorCodigo("HTL0001");
h->setNome(novoNome);
container.atualizar(h);

Inclui um novo hotel no container associado a um gerente.

Parameters
hotelPonteiro para o hotel a ser incluído
email_gerenteEmail do gerente proprietário do hotel
Returns
true se incluído com sucesso, false se o código já existe

O código do hotel deve ser único no sistema. Se já existir um hotel com o mesmo código (mesmo de outro gerente), a operação falha.

Se o gerente ainda não possui hotéis, cria uma nova entrada no map. Caso contrário, adiciona ao vector existente.

Exemplo de uso:

Hotel* h = new Hotel();
h->setCodigo(codigo);
h->setNome(nome);
if(container.incluir(h, "gerente@email.com")) {
cout << "Hotel incluído com sucesso";
}

Remove um hotel específico do container.

Parameters

Busca o hotel em todos os vetores de gerentes pelo código e remove. Se era o único hotel do gerente, remove a entrada do gerente do map.
