Detailed Description

Representa um endereço textual válido de 5 a 30 caracteres.

A classe Endereco é responsável por armazenar e validar endereços textuais conforme regras específicas de formatação. A validação inclui:

  • Comprimento entre 5 e 30 caracteres
  • Caracteres permitidos: letras (a-z, A-Z), dígitos (0-9), vírgula, ponto e espaço
  • Restrições de sequência de caracteres
  • Restrições para primeiro e último caractere

Exemplos de endereços válidos:

  • Rua das Flores, 123
  • Av. Principal, 1000
  • Praça da Sé, 25
