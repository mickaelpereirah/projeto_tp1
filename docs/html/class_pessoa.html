Detailed Description

Classe base que representa uma pessoa no sistema de gestão de hotéis.

Esta classe serve como base para outras entidades do sistema (Gerente e Hóspede). Contém os atributos básicos comuns a todas as pessoas: nome e email.

Atributos:

  • nome: Nome da pessoa (tipo Nome)
  • email: Email da pessoa (tipo Email)

Relacionamentos:

  • É especializada pelas classes Gerente e Hóspede
