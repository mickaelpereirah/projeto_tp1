Detailed Description

Representa a capacidade de um quarto, hotel ou entidade relacionada.

A classe Capacidade é responsável por armazenar e validar um valor inteiro que representa a capacidade (por exemplo, número de pessoas).

Regras típicas de validação (exemplo):

  • Deve ser um número positivo.
  • Deve ser menor ou igual ao limite definido pelo sistema.

O método setValor() realiza a validação e só atribui o valor se ele for válido. O método getValor() retorna o valor armazenado.
