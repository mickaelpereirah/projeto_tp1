Detailed Description

Interface Serviço Autenticação (ISA).

PROPÓSITO: Define operações padrão para autenticação e gerenciamento de sessões de usuários no sistema seguindo padrão UML.

ESTEREÓTIPO UML: <<interface>>

OPERAÇÕES ABSTRATAS:

  • +login(email : string, senha : string) : bool
  • +logout() : bool
  • +verificarSessao() : bool

RELACIONAMENTOS:

  • Realizada por: CSA (Central de Serviços de Autenticação)
  • Usada por: CAA (Caso de Uso Autenticação Administrador)
