\doxysubsection{Detailed Description}
Representa um valor monetário válido entre R\$ 1,00 e R\$ 1.\+000.\+000,00. 

A classe \doxylink{class_dinheiro}{Dinheiro} armazena valores monetários com precisão de centavos, utilizando internamente representação em centavos (inteiro) para evitar problemas de precisão com ponto flutuante.

Características principais\+:
\begin{DoxyItemize}
\item Intervalo\+: R\$ 1,00 a R\$ 1.\+000.\+000,00
\item Precisão\+: 2 casas decimais (centavos)
\item Armazenamento interno\+: valor em centavos (inteiro)
\item Formato de entrada/saída\+: ponto como separador decimal (ex\+: 150.\+75) 
\end{DoxyItemize}
