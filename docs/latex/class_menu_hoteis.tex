\doxysubsection{Detailed Description}
Classe responsável pelo gerenciamento de hotéis do gerente. 

Permite criar, listar, visualizar detalhes, editar e excluir hotéis. Em breve\+: selecionar hotel para gerenciar quartos, reservas e hóspedes.
