\doxysubsection{Detailed Description}
Atualiza dados de reservas cadastradas. 

{\bfseries{Permite alterar\+:}} \doxylink{class_data}{Data} Inicial, \doxylink{class_data}{Data} Final ~\newline
 {\bfseries{NÃO permite alterar\+:}} Código da \doxylink{class_reserva}{Reserva} (chave primária), \doxylink{class_quarto}{Quarto}, Hóspede ~\newline
 {\bfseries{Processo\+:}} Pergunta campo por campo se deseja alterar

{\bfseries{Regras de Negócio\+:}}
\begin{DoxyItemize}
\item \doxylink{class_data}{Data} inicial deve ser anterior à data final
\item Datas devem estar em formato válido
\item Período deve ser válido conforme disponibilidade do quarto 
\end{DoxyItemize}
