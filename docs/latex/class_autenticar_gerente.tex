\doxysubsection{Detailed Description}
Autentica gerentes no sistema. 

{\bfseries{Processo de Autenticação\+:}}
\begin{DoxyEnumerate}
\item Recebe email e senha
\item Busca gerente no container por email
\item Verifica se senha corresponde
\item Retorna gerente autenticado ou nullptr
\end{DoxyEnumerate}

{\bfseries{Segurança\+:}}
\begin{DoxyItemize}
\item Comparação direta de senha (sem criptografia no MVP)
\item \doxylink{class_email}{Email} como identificador único
\item Retorna nullptr em caso de falha (não expõe motivo)
\end{DoxyItemize}

{\bfseries{Casos de Uso\+:}}
\begin{DoxyItemize}
\item Login na camada de apresentação
\item Validação de permissões de acesso
\item Início de sessão do gerente 
\end{DoxyItemize}
