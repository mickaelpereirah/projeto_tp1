\doxysubsection{Detailed Description}
Representa um código identificador válido. 

A classe \doxylink{class_codigo}{Codigo} é responsável por armazenar e validar códigos utilizados no sistema para identificação de entidades.

As regras de validação típicas podem incluir\+:
\begin{DoxyItemize}
\item Formato específico (ex\+: AAA000)
\item Tamanho fixo ou variável
\item Caracteres permitidos (letras, números)
\end{DoxyItemize}

A implementação específica das regras de validação é definida no método \doxylink{class_codigo_ac5f23076fb06cb67121050014b861fa8}{validar()} privado.
