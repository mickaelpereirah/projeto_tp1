\doxysubsection{Detailed Description}
Representa um código identificador válido. 

A classe \doxylink{class_codigo}{Codigo} armazena e valida códigos alfanuméricos utilizados para identificar entidades no sistema (hotéis, reservas, etc.).

{\bfseries{FORMATO VÁ\+LIDO\+:}}
\begin{DoxyItemize}
\item Exatamente 10 caracteres
\item Cada caracter é letra (a-\/z) ou dígito (0-\/9)
\end{DoxyItemize}

{\bfseries{REGRAS DE VALIDAÇÃO\+:}}
\begin{DoxyEnumerate}
\item O código deve conter exatamente 10 caracteres
\item Cada caractere deve ser\+:
\begin{DoxyItemize}
\item Letra minúscula (a-\/z) OU
\item Dígito (0-\/9)
\end{DoxyItemize}
\item Não são permitidos\+:
\begin{DoxyItemize}
\item Letras maiúsculas (devem ser convertidas antes)
\item Caracteres especiais (-\/, \+\_\+, @, etc.)
\item Espaços em branco
\end{DoxyItemize}
\end{DoxyEnumerate}

{\bfseries{EXEMPLOS\+:}}
\begin{DoxyItemize}
\item ✅ Válido\+: "{}htl0001abc"{} (10 caracteres alfanuméricos)
\item ✅ Válido\+: "{}abc1234567"{} (mix de letras e números)
\item ✅ Válido\+: "{}0123456789"{} (apenas números)
\item ✅ Válido\+: "{}abcdefghij"{} (apenas letras)
\item ❌ Inválido\+: "{}\+HTL0001\+ABC"{} (contém letras maiúsculas)
\item ❌ Inválido\+: "{}abc123"{} (apenas 6 caracteres)
\item ❌ Inválido\+: "{}abc-\/123-\/456"{} (contém hífen)
\item ❌ Inválido\+: "{}abc 123 456"{} (contém espaços)
\end{DoxyItemize}

{\bfseries{NOTA DE IMPLEMENTAÇÃO\+:}} O sistema pode internamente converter letras maiúsculas para minúsculas ou exigir que o usuário forneça já em minúsculas.
