\doxysubsection{Detailed Description}
Classe que representa um gerente no sistema de gestão de hotéis. 

Esta classe herda de \doxylink{class_pessoa}{Pessoa} e adiciona atributos específicos para gerentes. Um gerente tem acesso ao sistema através de ramal e senha, podendo gerenciar hotéis, quartos, reservas e hóspedes.

Atributos herdados de \doxylink{class_pessoa}{Pessoa}\+:
\begin{DoxyItemize}
\item nome\+: \doxylink{class_nome}{Nome} do gerente
\item email\+: \doxylink{class_email}{Email} do gerente
\end{DoxyItemize}

Atributos específicos\+:
\begin{DoxyItemize}
\item ramal\+: \doxylink{class_ramal}{Ramal} do gerente para comunicação interna (tipo \doxylink{class_ramal}{Ramal})
\item senha\+: \doxylink{class_senha}{Senha} de acesso ao sistema (tipo \doxylink{class_senha}{Senha})
\end{DoxyItemize}
