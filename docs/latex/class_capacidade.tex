\doxysubsection{Detailed Description}
Representa a capacidade de um quarto, hotel ou entidade relacionada. 

A classe \doxylink{class_capacidade}{Capacidade} é responsável por armazenar e validar um valor inteiro que representa a capacidade (por exemplo, número de pessoas).

Regras típicas de validação (exemplo)\+:
\begin{DoxyItemize}
\item Deve ser um número positivo.
\item Deve ser menor ou igual ao limite definido pelo sistema.
\end{DoxyItemize}

O método \doxylink{class_capacidade_a8da632e9051370ca5f172911a839197a}{set\+Valor()} realiza a validação e só atribui o valor se ele for válido. O método \doxylink{class_capacidade_a51e9e105f9466a368541bff39017815b}{get\+Valor()} retorna o valor armazenado.
