\doxysubsection{Detailed Description}
Representa a capacidade de pessoas em um quarto. 

A classe \doxylink{class_capacidade}{Capacidade} armazena e valida o número de pessoas que um quarto pode acomodar. O valor deve estar dentro do conjunto permitido pelo sistema.

{\bfseries{FORMATO VÁ\+LIDO\+:}}
\begin{DoxyItemize}
\item Valores aceitos\+: 1, 2, 3 ou 4
\end{DoxyItemize}

{\bfseries{REGRAS DE VALIDAÇÃO\+:}}
\begin{DoxyItemize}
\item Apenas os valores 1, 2, 3 ou 4 são aceitos
\item Qualquer outro valor resulta em exceção invalid\+\_\+argument
\end{DoxyItemize}

{\bfseries{EXEMPLOS\+:}}
\begin{DoxyItemize}
\item ✅ Válido\+: 1 (quarto individual)
\item ✅ Válido\+: 2 (quarto duplo)
\item ✅ Válido\+: 3 (quarto triplo)
\item ✅ Válido\+: 4 (quarto quádruplo)
\item ❌ Inválido\+: 0 (não há quarto sem capacidade)
\item ❌ Inválido\+: 5 (acima do limite permitido)
\end{DoxyItemize}

O método \doxylink{class_capacidade_a8da632e9051370ca5f172911a839197a}{set\+Valor()} realiza a validação e lança exceção se inválido. O método \doxylink{class_capacidade_a51e9e105f9466a368541bff39017815b}{get\+Valor()} retorna o valor armazenado.
