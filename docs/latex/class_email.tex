\doxysubsection{Detailed Description}
Representa um endereço de email válido conforme padrões RFC. 

A classe \doxylink{class_email}{Email} é responsável por armazenar e validar endereços de email de acordo com as especificações RFC 5322. A validação inclui\+:
\begin{DoxyItemize}
\item Verificação da estrutura básica (parte local @ domínio)
\item Validação de caracteres permitidos em cada parte
\item Verificação do comprimento máximo (até 254 caracteres)
\item Validação do domínio (extensão, pontos, etc.)
\end{DoxyItemize}

Exemplos de emails válidos\+:
\begin{DoxyItemize}
\item \href{mailto:usuario@exemplo.com}{\texttt{usuario@exemplo.\+com}}
\item \href{mailto:nome.sobrenome@empresa.com.br}{\texttt{nome.\+sobrenome@empresa.\+com.\+br}}
\item \href{mailto:user+tag@domain.org}{\texttt{user+tag@domain.\+org}} 
\end{DoxyItemize}
