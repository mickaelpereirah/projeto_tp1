\doxysubsection{Detailed Description}
Lista hóspedes cadastrados no sistema. 

{\bfseries{Duas Abordagens de Listagem\+:}}

{\bfseries{Solução 1 -\/ Global (listar\+\_\+todos\+\_\+hospedes)\+:}}
\begin{DoxyItemize}
\item Lista TODOS os hóspedes do sistema
\item Independente de hotel ou reserva
\item Útil para visão administrativa geral
\end{DoxyItemize}

{\bfseries{Solução 2 -\/ Por \doxylink{class_hotel}{Hotel} (listar\+\_\+hospedes\+\_\+por\+\_\+hotel)\+:}}
\begin{DoxyItemize}
\item Lista apenas hóspedes COM RESERVAS em hotel específico
\item Filtra por quartos do hotel
\item Útil para gestão específica do hotel
\end{DoxyItemize}

{\bfseries{Método Padrão (listar\+\_\+hospedes)\+:}}
\begin{DoxyItemize}
\item Atualmente usa Solução 1 (todos os hóspedes)
\item Pode ser alterado para usar Solução 2 se necessário 
\end{DoxyItemize}
