\doxysubsection{Detailed Description}
Representa um número de telefone internacional válido. 

A classe \doxylink{class_telefone}{Telefone} é responsável por armazenar e validar números de telefone no formato internacional padrão. A validação inclui\+:
\begin{DoxyItemize}
\item Comprimento fixo de 14 caracteres total
\item Formato\+: \textquotesingle{}+\textquotesingle{} seguido de 13 dígitos
\item Primeiro caractere deve ser \textquotesingle{}+\textquotesingle{}
\item Os 13 caracteres restantes devem ser apenas dígitos (0-\/9)
\item Compatível com números internacionais reais
\end{DoxyItemize}

Exemplos de números válidos\+:
\begin{DoxyItemize}
\item +5561999999999 (Brasil\+: +55 61 99999-\/9999)
\item +5511999999999 (Brasil\+: +55 11 99999-\/9999)
\item +1234567890123 (Exemplo internacional) 
\end{DoxyItemize}
