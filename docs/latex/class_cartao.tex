\doxysubsection{Detailed Description}
Representa um número de cartão de crédito válido. 

A classe \doxylink{class_cartao}{Cartao} armazena e valida números de cartão de crédito utilizando o algoritmo de Luhn para garantir a integridade do dígito verificador.

{\bfseries{FORMATO VÁ\+LIDO\+:}}
\begin{DoxyItemize}
\item Exatamente 16 dígitos (0-\/9)
\item Presença de dígito verificador calculado segundo o algoritmo de Luhn
\end{DoxyItemize}

{\bfseries{REGRAS DE VALIDAÇÃO\+:}}
\begin{DoxyEnumerate}
\item O número deve conter exatamente 16 caracteres
\item Todos os caracteres devem ser dígitos numéricos (0-\/9)
\item O dígito verificador deve ser válido segundo o algoritmo de Luhn
\end{DoxyEnumerate}

{\bfseries{ALGORITMO DE LUHN\+:}}
\begin{DoxyEnumerate}
\item Da direita para esquerda, duplique cada segundo dígito
\item Se o resultado for maior que 9, subtraia 9
\item Some todos os dígitos (duplicados e não duplicados)
\item Se a soma for múltiplo de 10, o cartão é válido
\end{DoxyEnumerate}

{\bfseries{EXEMPLOS\+:}}
\begin{DoxyItemize}
\item ✅ Válido\+: 4556737586899855 (cartão Visa de teste)
\item ✅ Válido\+: 4111111111111111 (número de teste válido)
\item ✅ Válido\+: 5555555555554444 (Mastercard de teste)
\item ❌ Inválido\+: 1234567890123456 (falha no algoritmo de Luhn)
\item ❌ Inválido\+: 411111111111111 (apenas 15 dígitos)
\item ❌ Inválido\+: 4111-\/1111-\/1111-\/1111 (contém hífens)
\item ❌ Inválido\+: 4111 1111 1111 1111 (contém espaços)
\end{DoxyItemize}
