\doxysubsection{Detailed Description}
Representa um endereço textual válido de 5 a 30 caracteres. 

A classe \doxylink{class_endereco}{Endereco} é responsável por armazenar e validar endereços textuais conforme regras específicas de formatação. A validação inclui\+:
\begin{DoxyItemize}
\item Comprimento entre 5 e 30 caracteres
\item Caracteres permitidos\+: letras (a-\/z, A-\/Z), dígitos (0-\/9), vírgula, ponto e espaço
\item Restrições de sequência de caracteres
\item Restrições para primeiro e último caractere
\end{DoxyItemize}

Exemplos de endereços válidos\+:
\begin{DoxyItemize}
\item Rua das Flores, 123
\item Av. Principal, 1000
\item Praça da Sé, 25 
\end{DoxyItemize}
