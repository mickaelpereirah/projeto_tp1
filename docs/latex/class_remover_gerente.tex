\doxysubsection{Detailed Description}
Remove gerentes do sistema com remoção em cascata. 

{\bfseries{IMPORTANTE -\/ Remoção em Cascata\+:}}
\begin{DoxyItemize}
\item Remove GERENTE
\item Remove todos os HOTÉ\+IS do gerente
\item Remove todos os QUARTOS dos hotéis
\item Remove todas as RESERVAS dos quartos
\end{DoxyItemize}

{\bfseries{Impacto\+:}}
\begin{DoxyItemize}
\item Remoção permanente (não há recuperação no MVP)
\item Afeta toda hierarquia de dados
\item Hóspedes NÃO são removidos (apenas suas reservas)
\end{DoxyItemize}

{\bfseries{Segurança\+:}}
\begin{DoxyItemize}
\item Ação irreversível
\item Requer confirmação na camada de apresentação
\item \doxylink{class_gerente}{Gerente} só pode remover a si próprio (após login) 
\end{DoxyItemize}
