\doxysubsection{Detailed Description}
Remove hotéis do sistema com remoção em cascata. 

{\bfseries{Remoção em Cascata\+:}}
\begin{DoxyItemize}
\item Remove HOTEL
\item Remove todos os QUARTOS do hotel
\item Remove todas as RESERVAS dos quartos
\end{DoxyItemize}

{\bfseries{Regra de Negócio\+:}}
\begin{DoxyItemize}
\item \doxylink{class_hotel}{Hotel} com quartos e reservas pode ser removido
\item Todas as reservas são canceladas
\item Hóspedes NÃO são removidos (apenas suas reservas deste hotel)
\end{DoxyItemize}

{\bfseries{Impacto\+:}}
\begin{DoxyItemize}
\item Remove estrutura completa do hotel
\item Cancela todas as reservas ativas
\item Ação irreversível (sem recuperação no MVP) 
\end{DoxyItemize}
