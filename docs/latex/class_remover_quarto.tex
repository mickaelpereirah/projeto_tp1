\doxysubsection{Detailed Description}
Remove quartos do sistema com remoção em cascata. 

{\bfseries{Remoção em Cascata\+:}}
\begin{DoxyItemize}
\item Remove QUARTO
\item Remove todas as RESERVAS do quarto
\end{DoxyItemize}

{\bfseries{Regra de Negócio\+:}}
\begin{DoxyItemize}
\item \doxylink{class_quarto}{Quarto} com reservas ativas pode ser removido
\item Todas as reservas são canceladas automaticamente
\item \doxylink{class_hotel}{Hotel} permanece no sistema (apenas perde o quarto)
\end{DoxyItemize}

{\bfseries{Impacto\+:}}
\begin{DoxyItemize}
\item Remove capacidade de hospedagem do hotel
\item Cancela reservas futuras e ativas do quarto
\item Ação irreversível (sem recuperação no MVP) 
\end{DoxyItemize}
