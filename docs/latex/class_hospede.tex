\doxysubsection{Detailed Description}
Classe que representa um hóspede no sistema de gestão de hotéis. 

Esta classe herda de \doxylink{class_pessoa}{Pessoa} e adiciona atributos específicos para hóspedes. Um hóspede pode fazer reservas em hotéis e possui endereço e cartão de pagamento.

Atributos herdados de \doxylink{class_pessoa}{Pessoa}\+:
\begin{DoxyItemize}
\item nome\+: \doxylink{class_nome}{Nome} do hóspede
\item email\+: \doxylink{class_email}{Email} do hóspede
\end{DoxyItemize}
