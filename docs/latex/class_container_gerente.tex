\doxysubsection{Detailed Description}
Container para armazenar e gerenciar gerentes do sistema. 

Esta classe implementa um container para armazenamento em memória de objetos do tipo \doxylink{class_gerente}{Gerente}. Utiliza um unordered\+\_\+map com o email do gerente como chave, garantindo acesso rápido O(1) e unicidade de emails.

O container é responsável por\+:
\begin{DoxyItemize}
\item Armazenar gerentes de forma única (um email = um gerente)
\item Permitir busca eficiente por email
\item Gerenciar inclusão, remoção e atualização de gerentes
\item Manter a integridade dos dados (não permite emails duplicados)
\end{DoxyItemize}

IMPORTANTE\+: O container NÃO é responsável por liberar a memória dos objetos. A responsabilidade de delete dos ponteiros é de quem criou os objetos.
