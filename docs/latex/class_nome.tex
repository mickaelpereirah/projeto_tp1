\doxysubsection{Detailed Description}
Representa um nome pessoal válido de 5 a 20 caracteres. 

A classe \doxylink{class_nome}{Nome} é responsável por armazenar e validar nomes pessoais conforme regras específicas de formatação. A validação inclui\+:
\begin{DoxyItemize}
\item Comprimento entre 5 e 20 caracteres
\item Caracteres permitidos\+: apenas letras (a-\/z, A-\/Z) e espaços em branco
\item Primeira letra de cada termo deve ser maiúscula (A-\/Z)
\item Espaço em branco deve ser seguido por letra
\item Último caractere não pode ser espaço em branco
\end{DoxyItemize}

Exemplos de nomes válidos\+:
\begin{DoxyItemize}
\item Joao Silva
\item Maria Santos
\item Carlos Alberto
\item Ana Claudia 
\end{DoxyItemize}
