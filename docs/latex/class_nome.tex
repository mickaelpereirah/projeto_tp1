\doxysubsection{Detailed Description}
Representa um nome pessoal válido. 

A classe \doxylink{class_nome}{Nome} armazena e valida nomes pessoais seguindo regras específicas de formatação e capitalização.

{\bfseries{FORMATO VÁ\+LIDO\+:}}
\begin{DoxyItemize}
\item Texto com 5 a 20 caracteres
\end{DoxyItemize}

{\bfseries{CARACTERES PERMITIDOS\+:}}
\begin{DoxyItemize}
\item Letra (a-\/z ou A-\/Z)
\item Espaço em branco ( )
\end{DoxyItemize}

{\bfseries{REGRAS DE FORMATAÇÃO\+:}}
\begin{DoxyItemize}
\item Cada caracter pode ser letra (a-\/z ou A-\/Z) ou espaço em branco
\item Primeiro caracter de cada termo é letra maiúscula (A-\/Z)
\item Espaço em branco é seguido por letra
\item Último caracter não pode ser espaço em branco
\end{DoxyItemize}

{\bfseries{DEFINIÇÃO DE TERMO\+:}}
\begin{DoxyItemize}
\item Termo\+: sequência de letras separada por espaço
\item Exemplo\+: "{}\+Joao Silva"{} tem 2 termos\+: "{}\+Joao"{} e "{}\+Silva"{}
\item Cada termo deve começar com letra maiúscula
\end{DoxyItemize}

{\bfseries{EXEMPLOS\+:}}
\begin{DoxyItemize}
\item ✅ Válido\+: "{}\+Joao Silva"{} (5 caracteres, 2 termos capitalizados)
\item ✅ Válido\+: "{}\+Maria Santos"{} (12 caracteres)
\item ✅ Válido\+: "{}\+Carlos Alberto Souza"{} (20 caracteres, 3 termos)
\item ✅ Válido\+: "{}\+Ana C"{} (5 caracteres, inicial permitida)
\item ✅ Válido\+: "{}\+Pedro De Almeida"{} ("{}\+De"{} capitalizado)
\item ❌ Inválido\+: "{}\+João"{} (apenas 4 caracteres, mínimo é 5)
\item ❌ Inválido\+: "{}\+Nome Muito Grande Demais"{} (mais de 20 caracteres)
\item ❌ Inválido\+: "{}joao silva"{} (não começa com maiúscula)
\item ❌ Inválido\+: "{}\+Joao silva"{} (segundo termo não começa com maiúscula)
\item ❌ Inválido\+: "{}\+JOAO SILVA"{} (todas maiúsculas, não segue padrão)
\item ❌ Inválido\+: "{}\+Joao Silva "{} (termina com espaço)
\item ❌ Inválido\+: "{} Joao Silva"{} (começa com espaço)
\item ❌ Inválido\+: "{}\+Joao  Silva"{} (espaços consecutivos)
\item ❌ Inválido\+: "{}\+Joao123"{} (contém dígito)
\item ❌ Inválido\+: "{}\+Joao-\/\+Silva"{} (contém hífen)
\item ❌ Inválido\+: "{}\+João Silva"{} (contém acentuação)
\end{DoxyItemize}

{\bfseries{VALIDAÇÕ\+ES ESPECÍ\+FICAS\+:}}
\begin{DoxyItemize}
\item Comprimento entre 5 e 20 caracteres (inclusive)
\item Apenas letras (a-\/z, A-\/Z) e espaços permitidos
\item Primeiro caractere de cada termo\+: maiúscula (A-\/Z)
\item Após o primeiro caractere de cada termo\+: minúsculas (a-\/z)
\item Espaço sempre seguido por letra maiúscula (início de novo termo)
\item Não pode começar ou terminar com espaço
\item Não permite espaços consecutivos 
\end{DoxyItemize}
