\doxysubsection{Detailed Description}
Classe responsável pelo gerenciamento de reservas de um hotel. 

Esta classe fornece interface completa para operações CRUD de reservas, incluindo validações de disponibilidade, cálculo automático de valores e associação com quartos e hóspedes. Gerencia todo o ciclo de vida de uma reserva desde a criação até a exclusão.

Características principais\+:
\begin{DoxyItemize}
\item Validação de conflito de datas entre reservas
\item Cálculo automático do valor total baseado no período
\item Recálculo automático quando datas são alteradas
\item Verificação de existência de quartos e hóspedes
\item Exibição de informações contextualizadas (quarto, hóspede) 
\end{DoxyItemize}
