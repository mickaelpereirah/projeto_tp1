\doxysubsection{Detailed Description}
Remove reservas do sistema. 

{\bfseries{Remoção Simples\+:}}
\begin{DoxyItemize}
\item Remove apenas a RESERVA
\item NÃO remove \doxylink{class_quarto}{Quarto} (fica disponível)
\item NÃO remove Hóspede (mantém cadastro)
\end{DoxyItemize}

{\bfseries{Regra de Negócio\+:}}
\begin{DoxyItemize}
\item \doxylink{class_reserva}{Reserva} pode ser cancelada a qualquer momento
\item \doxylink{class_quarto}{Quarto} volta a ficar disponível após remoção
\item Hóspede mantém outras reservas (se houver)
\end{DoxyItemize}

{\bfseries{Uso Principal\+:}}
\begin{DoxyItemize}
\item Cancelamento de reservas
\item Liberação de quartos
\item Correção de erros de cadastro 
\end{DoxyItemize}
