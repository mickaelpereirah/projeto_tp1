\doxysubsection{Detailed Description}
Módulos responsáveis pela interface com o usuário. 

Esta camada contém todos os componentes que interagem diretamente com o usuário, incluindo menus, formulários e controles de navegação.

{\bfseries{Componentes\+:}}
\begin{DoxyItemize}
\item Interface Principal (main)
\item Casos de Uso (Menus específicos)
\item Interfaces abstratas
\end{DoxyItemize}

{\bfseries{Responsabilidades\+:}}
\begin{DoxyItemize}
\item Capturar entrada do usuário
\item Validar dados de entrada
\item Exibir resultados e mensagens
\item Controlar fluxo de navegação
\item Delegar operações para camada de serviços 
\end{DoxyItemize}
