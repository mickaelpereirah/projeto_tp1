\doxysubsection{Detailed Description}
Representa uma senha de acesso segura. 

A classe \doxylink{class_senha}{Senha} armazena e valida senhas seguindo regras rigorosas de segurança para garantir senhas fortes.

{\bfseries{FORMATO VÁ\+LIDO\+:}}
\begin{DoxyItemize}
\item Exatamente 5 caracteres
\end{DoxyItemize}

{\bfseries{CARACTERES PERMITIDOS\+:}}
\begin{DoxyItemize}
\item Letra minúscula (a-\/z)
\item Letra maiúscula (A-\/Z)
\item Dígito (0-\/9)
\item Caracter especial\+: ! "{} \# \$ \% \& ?   $<$strong$>$\+REGRAS DE COMPOSIÇÃ\+O\+:$<$/strong$>$ -\/ Letra NÃ\+O pode ser seguida por letra -\/ Dígito NÃ\+O pode ser seguido por dígito -\/ Deve existir pelo menos\+:   -\/ Uma letra minúscula (a-\/z)   -\/ Uma letra maiúscula (\+A-\/\+Z)   -\/ Um dígito (0-\/9)   -\/ Um caracter especial (! "{} \# \$ \% \& ?)
\end{DoxyItemize}

{\bfseries{EXEMPLOS\+:}}
\begin{DoxyItemize}
\item ✅ Válido\+: "{}a\+A1\#b"{} (tem tudo\+: minúscula, maiúscula, dígito, especial, alternados)
\item ✅ Válido\+: "{}1a\+B\%2"{} (dígito, letra, letra, especial, dígito)
\item ✅ Válido\+: "{}\+A1b!\+C"{} (alterna tipos)
\item ✅ Válido\+: "{}9\#\+Aa8"{} (todos os requisitos atendidos)
\item ❌ Inválido\+: "{}a\+A1\#"{} (apenas 4 caracteres, precisa de 5)
\item ❌ Inválido\+: "{}a\+A1\#bc"{} (6 caracteres, precisa de exatamente 5)
\item ❌ Inválido\+: "{}a\+B1\#2"{} (dígito seguido de dígito\+: "{}1\#2"{})
\item ❌ Inválido\+: "{}ab1\#2"{} (letra seguida de letra\+: "{}ab"{})
\item ❌ Inválido\+: "{}\+AB1\#2"{} (letra seguida de letra\+: "{}\+AB"{})
\item ❌ Inválido\+: "{}1\#2\$3"{} (falta letra minúscula)
\item ❌ Inválido\+: "{}a1\#2\$"{} (falta letra maiúscula)
\item ❌ Inválido\+: "{}a\+A\#\$\%"{} (falta dígito)
\item ❌ Inválido\+: "{}a\+A123"{} (falta caracter especial)
\item ❌ Inválido\+: "{}a\+A1@2"{} (@ não é caracter especial permitido)
\end{DoxyItemize}

{\bfseries{DETALHES DAS REGRAS\+:}}


\begin{DoxyEnumerate}
\item {\bfseries{Alternância obrigatória\+:}}
\begin{DoxyItemize}
\item Após letra (a-\/z ou A-\/Z)\+: deve vir dígito ou especial
\item Após dígito (0-\/9)\+: deve vir letra ou especial
\item Após especial\+: pode vir qualquer tipo
\end{DoxyItemize}
\item {\bfseries{Requisitos de presença\+:}}
\begin{DoxyItemize}

\end{DoxyItemize}
