\doxysubsection{Detailed Description}
Representa uma senha segura de exatamente 5 caracteres. 

A classe \doxylink{class_senha}{Senha} é responsável por armazenar e validar senhas seguras conforme regras rigorosas de segurança. A validação inclui\+:
\begin{DoxyItemize}
\item Comprimento fixo de exatamente 5 caracteres
\item Caracteres permitidos\+: letras (a-\/z, A-\/Z), dígitos (0-\/9) e caracteres especiais (!"{}\#\$\%\&?) -\/ Pelo menos uma letra minúscula (a-\/z) -\/ Pelo menos uma letra maiúscula (\+A-\/\+Z) -\/ Pelo menos um dígito (0-\/9) -\/ Pelo menos um caractere especial (!"{}\#\$\%\&?)
\item Letra não pode ser seguida por letra
\item Dígito não pode ser seguido por dígito
\end{DoxyItemize}

Exemplos de senhas válidas\+:
\begin{DoxyItemize}
\item A1b\#2
\item p@\+Q3r
\item X9!y8 
\end{DoxyItemize}
