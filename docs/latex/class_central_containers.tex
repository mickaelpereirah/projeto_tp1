\doxysubsection{Detailed Description}
Classe central para gerenciar todos os containers do sistema. 

Esta classe implementa o padrão Singleton para fornecer acesso centralizado a todos os containers de armazenamento de dados do sistema. Cada container gerencia uma entidade específica do domínio (Reservas, Hóspedes, Hotéis, etc.).

Os containers são estáticos e únicos durante toda a execução do programa, garantindo que todos os módulos acessem a mesma instância de dados.

Arquitetura\+:
\begin{DoxyItemize}
\item Camada de Serviço\+: acessa containers através desta classe
\item Camada de Apresentação\+: NÃO acessa diretamente, usa serviços
\item Armazenamento\+: em memória (RAM) durante execução do programa
\end{DoxyItemize}

Responsabilidades\+:
\begin{DoxyItemize}
\item Fornecer acesso único aos containers
\item Garantir inicialização correta dos containers
\item Manter singleton dos containers durante execução
\end{DoxyItemize}

IMPORTANTE\+: Os dados são perdidos quando o programa é encerrado. Não há persistência em banco de dados ou arquivo.
