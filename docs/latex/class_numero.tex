\doxysubsection{Detailed Description}
Representa um número inteiro válido de 1 a 999. 

A classe \doxylink{class_numero}{Numero} é responsável por armazenar e validar números inteiros dentro de um intervalo específico. A validação inclui\+:
\begin{DoxyItemize}
\item Intervalo de valores\+: 1 a 999 (inclusive)
\item Apenas valores inteiros são aceitos
\item Formatação de saída com 3 dígitos (ex\+: 001, 083, 999)
\end{DoxyItemize}

A classe é útil para representar números que requerem formatação consistente, como códigos, identificadores ou valores numéricos com tamanho fixo.

Exemplos de valores válidos\+:
\begin{DoxyItemize}
\item 1 (formatado como 001)
\item 83 (formatado como 083)
\item 999 (formatado como 999) 
\end{DoxyItemize}
