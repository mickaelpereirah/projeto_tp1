\doxysubsection{Detailed Description}
Classe que representa uma reserva no sistema de gestão de hotéis. 

Esta classe encapsula as informações de uma reserva, incluindo datas de chegada e partida, valor total e código de identificação.

Atributos\+:
\begin{DoxyItemize}
\item chegada\+: \doxylink{class_data}{Data} de chegada da reserva (tipo \doxylink{class_data}{Data})
\item partida\+: \doxylink{class_data}{Data} de partida da reserva (tipo \doxylink{class_data}{Data})
\item valor\+: Valor total da reserva (tipo \doxylink{class_dinheiro}{Dinheiro})
\item codigo\+: Código único de identificação da reserva (tipo \doxylink{class_codigo}{Codigo} -\/ chave primária)
\end{DoxyItemize}

Relacionamentos\+:
\begin{DoxyItemize}
\item É feita por Hóspede (0\+:\texorpdfstring{$\ast$}{*} para 1)
\item \doxylink{class_reserva}{Reserva} \doxylink{class_quarto}{Quarto} (1 para 0\+:\texorpdfstring{$\ast$}{*})
\end{DoxyItemize}

Regras de negócio\+:
\begin{DoxyItemize}
\item Não podem ocorrer conflitos entre reservas do mesmo quarto
\item \doxylink{class_data}{Data} de partida deve ser posterior à data de chegada 
\end{DoxyItemize}
