\doxysubsection{Detailed Description}
Representa um ramal telefônico de quarto. 

A classe \doxylink{class_ramal}{Ramal} armazena e valida números de ramais telefônicos associados a quartos de hotel.

{\bfseries{FORMATO VÁ\+LIDO\+:}}
\begin{DoxyItemize}
\item Valores de 00 a 50
\end{DoxyItemize}

{\bfseries{REGRAS DE VALIDAÇÃO\+:}}
\begin{DoxyItemize}
\item Valor mínimo\+: 0 (representado como 00)
\item Valor máximo\+: 50
\item Armazenado como inteiro (0 a 50)
\item Exibido com 2 dígitos (00, 01, ..., 50)
\end{DoxyItemize}

{\bfseries{EXEMPLOS\+:}}
\begin{DoxyItemize}
\item ✅ Válido\+: 0 (ramal 00)
\item ✅ Válido\+: 10 (ramal 10)
\item ✅ Válido\+: 50 (ramal máximo)
\item ❌ Inválido\+: -\/1 (valor negativo)
\item ❌ Inválido\+: 51 (acima do máximo)
\item ❌ Inválido\+: 100 (muito acima do máximo)
\end{DoxyItemize}

{\bfseries{NOTA DE USO\+:}}
\begin{DoxyItemize}
\item Ramais podem ser repetidos em quartos diferentes
\item \doxylink{class_ramal}{Ramal} 00 pode indicar que não há ramal associado
\item Usado para comunicação interna do hotel 
\end{DoxyItemize}
