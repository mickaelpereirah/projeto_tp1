\doxysubsection{Detailed Description}
Representa um número de ramal telefônico válido de 0 a 50. 

A classe \doxylink{class_ramal}{Ramal} é responsável por armazenar e validar números de ramal telefônico dentro de um intervalo específico. A validação inclui\+:
\begin{DoxyItemize}
\item Intervalo de valores\+: 0 a 50 (inclusive)
\item Apenas valores inteiros são aceitos
\end{DoxyItemize}

A classe é útil para sistemas que necessitam gerenciar ramais telefônicos com validação de intervalo e formatação consistente.

Exemplos de valores válidos\+:
\begin{DoxyItemize}
\item 0 (formatado como 0)
\item 5 (formatado como 5)
\item 50 (formatado como 50) 
\end{DoxyItemize}
