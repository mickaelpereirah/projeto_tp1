\doxysubsection{Detailed Description}
Serviço responsável pelo cadastro de hóspedes no sistema. 

Gerencia o cadastro de hóspedes, coletando dados pessoais e de pagamento.

{\bfseries{DADOS COLETADOS\+:}}
\begin{DoxyItemize}
\item \doxylink{class_nome}{Nome} (5-\/20 caracteres)
\item \doxylink{class_email}{Email} (único, chave primária)
\item Endereço (5-\/30 caracteres)
\item Cartão (16 dígitos + Luhn)
\end{DoxyItemize}

{\bfseries{VALIDAÇÕ\+ES\+:}}
\begin{DoxyItemize}
\item \doxylink{class_email}{Email} único no sistema
\item Cartão com dígito verificador válido (Luhn)
\item Todos os domínios validados individualmente 
\end{DoxyItemize}
