\doxysubsection{Detailed Description}
Container para armazenar e gerenciar hotéis do sistema. 

Esta classe implementa um container para armazenamento em memória de objetos do tipo \doxylink{class_hotel}{Hotel}. Utiliza uma estrutura de unordered\+\_\+map onde a chave é o email do gerente e o valor é um vector de hotéis, permitindo que cada gerente possua múltiplos hotéis (relacionamento 1\+:N).

Características principais\+:
\begin{DoxyItemize}
\item Um gerente pode ter vários hotéis
\item Cada hotel possui código único no sistema
\item Busca por gerente retorna todos os seus hotéis
\item Busca por código retorna hotel específico
\item Permite remoção de hotel individual ou de todos os hotéis de um gerente
\end{DoxyItemize}
